# Online Retail Sales Analysis
Exploratory Data Analysis of Online Retail Sales Dataset

## Project Overview
This project analyzes retail transaction data to identify sales trends, top products, and customer behavior.

## Dataset
Online Retail Dataset from UCI Machine Learning Repository.

## Tools
Python  
Pandas  
Matplotlib  
Seaborn  

## Key Analysis
- Top selling products
- Revenue by country
- Monthly sales trends

## Key Insights
- UK generates the highest revenue
- Certain products dominate sales
- Sales peak during holiday seasons
